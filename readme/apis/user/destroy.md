# Userデータ削除

登録済みUserデータに対して、idを指定してデータを削除します。

---

**URL** : `/api/user/{id}/`

**メソッド** : `DELETE`

**認証** : 必須（`auth:sanctum`）

**権限** : 不要

**データ制約** : なし

**ヘッダー制約** : `Authorization: Bearer {token}`  

---

## 成功レスポンス

**条件** : データの削除が正常に完了。

**コード** : `200 OK`

## エラーレスポンス

**条件** : 適切なidが指定されていない。

**コード** : `404 Not Found`
