# Productデータ取得

登録済みProductデータに対して、`id` を指定してデータを取得します。

---

**URL** : `/api/products/{id}`

**メソッド** : `GET`

**認証** : 必須（`auth:sanctum`）

**権限** : 不要

**データ制約** : なし

**ヘッダー制約** : なし

---

## 成功レスポンス

### データ取得成功

**コード** : `200 OK`

## エラーレスポンス

### 不正なid

**条件** : 適切なidが指定されていない。

**コード** : `404 Not Found`
