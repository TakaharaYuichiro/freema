# 認証ユーザーの全てのお気に入り登録データ取得

認証ユーザーのお気に入り登録データに対して、`product_id` を指定してデータを取得します。

---

**URL** : `/api/get-favorites/{product_id}`

**メソッド** : `GET`

**認証** : 必須（`auth:sanctum`）

**権限** : 不要

**データ制約** : なし

**ヘッダー制約** : `Authorization: Bearer {token}`  

---

## 成功レスポンス

### データ取得成功

**コード** : `200 OK`

## エラーレスポンス

### 不正なid

**条件** : 適切なidが指定されていない。

**コード** : `404 Not Found`
